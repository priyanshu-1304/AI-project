Overview:-
This project is a Smart Attendance System built using Python and OpenCV.
It captures a user's image through a webcam and records attendance with the name and timestamp in a CSV file. The captured image is also saved for verification.

Technologies Used:-
Python
OpenCV
Pandas
Datetime Module
OS Module

How It Works:-
User enters their name.
Webcam captures the image.
Image is enhanced using preprocessing.
Attendance is recorded in attendance.csv.
Captured image is saved in a folder.

Requirements:-
Python 3.x
OpenCV (pip install opencv-python)
Pandas (pip install pandas)
